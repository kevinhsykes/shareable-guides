# Claude Code Instructions — shareable-guides

## What this repo is

Collection of setup guides for Kevin Sykes' projects. Every guide is safe to share publicly — no API keys, passwords, or personal account details.

## Structure

- `PC_SETUP.md` — prerequisite: fresh Windows PC setup (Python, Git, VS Code, Claude Code)
- One guide per project: `golisano-study-tools.md`, `inventory.md`, `papervault.md`, `local-ai-lab.md`, `sfe-speaker-notes.md`

## Rules

1. **Never include secrets, API keys, or account-specific values.** Use placeholders like `YOUR_KEY_HERE`.
2. **These are copies of each project's SETUP.md.** If the source project's setup changes, update the copy here too.
3. **Keep guides self-contained.** A reader should be able to follow any guide without reading other files (except PC_SETUP.md as a prerequisite).
4. **Correct school name:** Golisano Institute for Business & Entrepreneurship (never "RIT Golisano").
